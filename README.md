# 🧾 영수증 관리

영수증 사진을 업로드하면 OCR로 자동 인식하여 지출 내역을 정리해주는 웹 애플리케이션입니다.

## 주요 기능

- **영수증 OCR 인식**: 영수증 사진을 업로드하면 Tesseract.js로 텍스트를 자동 추출
- **자동 분류**: 가게명 기반으로 카테고리 자동 분류 (식료품, 외식, 교통, 쇼핑 등 9개 카테고리)
- **달력 뷰**: 날짜별 영수증을 달력에서 한눈에 확인
- **대시보드**: 월별 총 지출, 카테고리별 비율, 지출 추이 차트
- **검색/필터**: 가게명, 품목으로 검색 및 카테고리별 필터링
- **반응형 디자인**: PC와 모바일 모두 지원

## 기술 스택

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Tesseract.js (OCR)
- date-fns
- LocalStorage (데이터 저장)

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`으로 접속하세요.
